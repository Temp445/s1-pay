# Face Recognition Models

This directory contains the face-api.js models required for face recognition.

## Required Models

The following models should be downloaded and placed in this directory:

- `face_landmark_68_model-shard1`
- `face_landmark_68_model-weights_manifest.json`
- `face_recognition_model-shard1`
- `face_recognition_model-shard2`
- `face_recognition_model-weights_manifest.json`
- `ssd_mobilenetv1_model-shard1`
- `ssd_mobilenetv1_model-shard2`
- `ssd_mobilenetv1_model-weights_manifest.json`
- `tiny_face_detector_model-shard1`
- `tiny_face_detector_model-weights_manifest.json`

## Downloading Models

These models can be downloaded from the face-api.js GitHub repository:
https://github.com/justadudewhohacks/face-api.js/tree/master/weights

## Usage

The models are automatically loaded when the face recognition system is initialized.
